# customer-churn-analysis
An end-to-end data analytics project analyzing customer churn trends and retention factors using Python, SQL/Snowflake, and Power BI..

Project Title
Customer Churn Analysis Dashboard

Objective
Analyze customer behavior and identify factors affecting churn.

Tools Used

Python (Pandas, NumPy)
SQL
Snowflake
Power BI

Workflow

Downloaded raw data from Kaggle
Cleaned and preprocessed data using Python
Performed SQL/Snowflake analysis
Built interactive dashboard in Power BI


Key Insights

Month-to-month contracts have highest churn
High monthly charges increase churn
Fiber optic users show higher churn
Low-tenure customers are at high risk
Dashboard Features
Churn rate KPI
Contract impact analysis
Payment method analysis
Tenure analysis
Internet service analysis
